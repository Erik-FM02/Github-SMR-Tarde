Acerca de Mí

Soy un desarrollador web junior en este tipo de tecnologías. Pero desde la infancia aprendí las nociones de la programación y la informática en general.
Aprendí de forma autodidacta y en mi ciclo de formación las tecnologías de HTML, CSS y JavaScript. También he realizado cursos IT en varias plataformas (OpenWebinars o Udemy) para mejorar mi dominio en el desarrollo web.
Actualmente estoy perfeccionando mi soltura para conseguir un nivel apto para un entorno laboral. Mi plan de formación a medio y largo plazo, pasa por aprender nuevas habilidades para perfiles tipo Front-end, Back-end y Full Stack developer. Así como aprender a usar tecnologías tipo Angular, React, Docker, PHP o SQL, lenguajes de programación como Python y Java, entre otros.
